## Introduction

The Jordan Shoes website is a single-page application designed to showcase different models of Jordan shoes. It provides users with information about each shoe model and allows them to explore the shoe details.

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Installation

To run this project locally, follow these steps:

1. Clone the repository.
2. Navigate to the project directory.
3. Open the `index.html` file in your preferred web browser.
